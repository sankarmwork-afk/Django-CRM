# Django CRM System

A simple CRM (Customer Relationship Management) web application built using Django, Bootstrap, and PostgreSQL/MySQL.

## Features

- User Registration
- User Login & Logout
- Authentication System
- Add Customer Records
- Update Records
- Delete Records
- View Customer Details
- Django Admin Panel
- Responsive Bootstrap UI

## Technologies Used

- Python
- Django
- Bootstrap
- HTML
- CSS
- PostgreSQL / mysql
- Git & GitHub
